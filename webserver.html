    • Run the Web Server:
  • Run TinyWeb.exe from the extracted folder.
  • Ensure that the server is running and ready to accept requests.
    • Check Access through the Local Browser:
  • Open a web browser and enter the following in the address bar: http://localhost.
  • Verify that you see the content of your test page.
    • Determine the IP Address of Your Computer:
  • Open the command prompt (cmd).
  • Enter the command: ipconfig /all | findstr 'IPv4'.
  • Note down your IPv4 address.
    • Check Access from Another Computer:
  • On another computer, open a web browser.
  • Enter the IP address of the computer running the web server in the browser's address bar, for example: http://192.168.x.x (where 192.168.x.x is your IPv4 address).
  • Ensure that you can view the test page through the browser on the other computer.

    • By following these steps, you have tested the functionality of your lightweight web server and confirmed successful access through both the local browser and another computer via the IP address.
